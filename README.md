# 🎓 Placement Package Predictor

A Machine Learning web application that predicts a student's expected placement package based on their CGPA. The application is built using **Python**, **Flask**, **Scikit-learn**, **HTML**, **CSS**, and **JavaScript**.

---

## 🚀 Features

- Predict placement package using a trained Linear Regression model
- Interactive and responsive web interface
- Flask backend with REST API
- Real-time prediction without page reload
- Easy to deploy and extend

---

## 🛠️ Tech Stack

- Python
- Flask
- Scikit-learn
- NumPy
- HTML5
- CSS3
- JavaScript

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/your-username/placement-package-predictor.git
```

### Navigate to the project

```bash
cd placement-package-predictor
```

### Create a virtual environment

```bash
python -m venv venv
```

### Activate the virtual environment

**Windows**

```bash
venv\Scripts\activate
```

**Linux / macOS**

```bash
source venv/bin/activate
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Run the application

```bash
python app.py
```

Open your browser and visit:

```
http://127.0.0.1:5000
```
